# 📌 Visão Geral

Este portfólio foi criado com foco em performance, organização, responsividade e um design moderno com estética neon suave.

Além de apresentar meus projetos e habilidades, o site também funciona como uma forma prática de aplicar e evoluir meus conhecimentos em desenvolvimento front-end.

O portfólio inclui:

- 🏠 Página Home
- 📂 Página de Projetos
- 🧠 Página de Habilidades
- 👤 Página Sobre Mim
- 📱 Menu Responsivo
- ⚡ Navegação fluida e animações suaves
- 💻 Layout 100% responsivo
